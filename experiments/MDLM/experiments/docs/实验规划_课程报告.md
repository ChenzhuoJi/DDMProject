| 特性 | AR (自回归) | MDLM (扩散) |
|------|:----------:|:----------:|

## 与《随机过程》课程的联系

| 实验现象 | 课程概念 | 论文对应 |
|---------|---------|---------|
| t=1 → t=0 逐步去噪 | **非时齐 Markov 链** | 前向过程 q(z_t\|x) |
| MASK 为吸收态 | **吸收态 Markov 链** | 吸收态扩散 |
| 连续时间极限 | **CTMC** | 连续时间 ELBO |
| noise schedule α_t | **转移速率矩阵** | σ(t) 噪声调度 |
| ddpm_cache 复用 | **Markov 性**（无后效性） | 条件独立假设 |
| 不同 seed → 不同样本 | **随机过程的样本路径** | 反向采样 |

\chapter{Masked Diffusion Model 复现}

\section{实验配置}
  硬件: RTX PRO 6000 (Blackwell, 102GB)
  推理模型: kuleshov-group/mdlm-no_flashattn-fp32-owt
  训练模型: DiT Tiny (本地 backbone)
  环境限制: Blackwell 架构无法安装 flash-attn →
            推理用 HF 官方 fp32 版(Colab T4)
            训练用本地 DiT + 原生 attention 回退(RTX PRO 6000)

\section{扩散过程可视化}
  \subsection{多随机种子下的去噪路径}
    seed 42 (PPL 52, 体育财经) / 123 (PPL 41, 教育引用) / 999 (PPL 8, 数字)
    → 同一初始全 [MASK] 状态经不同随机路径得到不同文本
    → 随机过程视角：非时齐 Markov 链的样本路径

  \subsection{采样步数对生成质量的影响}
    128步(PPL 137, 3.5s) → 500步(PPL 79, 7.2s) → 1000步(PPL 52, 9.5s)
    PPL 随步数增加而降低，质量单调提升
    → 随机过程视角：时间离散化精度与生成质量的关系

\section{采样算法效率分析}
  \subsection{三种采样器的原理对比}
    ddpm_cache: 缓存 p(x₀|xₜ)，token 不变时跳过前向计算
    ddpm: 标准 ancestral 采样，每步跑网络
    analytic: SEDD 式解析采样，使用得分函数
  \subsection{速度与质量评测}
    ddpm_cache 9.6s / ddpm 31.8s / analytic 36.5s (同 seed PPL 均 52)
    ddpm_cache 比 ddpm 快 3.3 倍，比 analytic 快 3.8 倍
    三者同 seed 输出完全一致 → 等价性验证通过
    → 随机过程视角：Markov 性(无后效性)使缓存成为可能

\section{text8 小规模训练}
  \subsection{训练配置}
    模型: DiT Tiny (8层, 512隐藏, 8头, ~5M参数)
    数据集: text8 (字符级, 27 token, ~100MB Wikipedia)
    超参数: 10,000 steps, Batch size 256, lr 3e-4, LogLinear 噪声
    训练时间: ~2 小时 (RTX PRO 6000, 无 flash-attn)
  \subsection{Loss 收敛曲线与分析}
    val/nll: 2.903 (step 250) → 1.327 (step 9980, best)
    Loss 持续下降，未完全收敛(论文 1M steps 达 ~1.4 BPC)
    因 Blackwell 无 flash-attn，训练速度受限(原生 attention 回退)
    10K steps 模型生成以 [UNK]/[PAD] 为主，尚未形成有效文本
    → 随机过程视角：ELBO 作为变分推断目标函数的有效性

\section{长文本生成与半自回归扩展}
  length=1024, 87.3s, PPL 44, 政治评论风格
  半自回归(stride)机制：逐步扩展上下文，突破 1024 长度限制
  展示扩散模型的灵活性和可扩展性

\section{实验总结}
  核心结论汇总 + 随机过程概念对照表
  (非时齐 Markov 链/吸收态/样本路径/CTMC/ELBO/转移速率矩阵)

## 具体命令

### 实验一：多维可视化

```bash
# seed 对比
for seed in 42 123 999; do
  python main.py mode=sample_eval \
    eval.checkpoint_path=kuleshov-group/mdlm-owt \
    data=openwebtext-split model.length=256 \
    sampling.predictor=ddpm_cache sampling.steps=128 \
    loader.eval_batch_size=1 sampling.num_sample_batches=1 \
    sampling.visualize=True seed=$seed backbone=hf_dit
done

# 步数对比
for steps in 128 500 1000; do
  python main.py mode=sample_eval \
    eval.checkpoint_path=kuleshov-group/mdlm-owt \
    data=openwebtext-split model.length=256 \
    sampling.predictor=ddpm_cache sampling.steps=$steps \
    loader.eval_batch_size=1 sampling.num_sample_batches=1 \
    sampling.visualize=True backbone=hf_dit
done
```

### 实验二：采样器对比

```bash
for predictor in ddpm_cache ddpm analytic; do
  time python main.py mode=sample_eval \
    eval.checkpoint_path=kuleshov-group/mdlm-owt \
    data=openwebtext-split model.length=256 \
    sampling.predictor=$predictor sampling.steps=1000 \
    loader.eval_batch_size=1 sampling.num_sample_batches=1 \
    backbone=hf_dit
done
```
