## 可视化数据（`experiments/snapshots/`）

扩散过程的逐行快照，展示从 `t=1.0`（全 MASK）到 `t=0.0`（完整文本）的去噪过程。
每份快照包含 **26 个时间点**（覆盖率潜伏期 9 个 + 主去噪 9 个 + 微调期 8 个），
文本内容**无截断**（`save_snapshots_to_file` 的 `max_length=0`）。

### 新版快照（推理模型：mdlm-no_flashattn-fp32-owt，Colab T4）

| 文件 | 大小 | 说明 |
|:----|:---:|:-----|
| **snapshot_seed42.txt** | 15 KB | seed=42, 1000步, ddpm_cache, length=256 |
| **snapshot_seed123.txt** | 18 KB | seed=123, 同上 |
| **snapshot_seed999.txt** | 11 KB | seed=999, 同上 |
| **snapshot_steps128.txt** | 17 KB | 步数对比：128步, seed=42 |
| **snapshot_steps500.txt** | 16 KB | 步数对比：500步, seed=42 |
| **snapshot_steps1000.txt** | 15 KB | 步数对比：1000步, seed=42 |
| **snapshot_length1024.txt** | 70 KB | 长文本：length=1024, seed=123 |

### 历史版本（旧模型/存档）

| 文件 | 说明 |
|:----|:-----|
| `legacy_seed42/123/999.txt` | 旧模型 `mdlm-owt`（需 flash_attn）的 128 步结果，供存档参考 |
| `legacy_steps500.txt` | 旧模型 500 步对比 |
| `legacy_text8.txt` | text8 tiny 模型生成（本地训练） |
| `v1_*.txt` | 正确模型的首批可视化结果（256 token, 13 时间点） |
| `v2_*.txt` | 正确模型的第二批可视化结果 |
| `v3_*.txt` | 正确模型的第三批可视化结果 |

### 文件格式

```
======================================================================
扩散逆向过程 (t=1.0 → t=0.0)
======================================================================
t=1.000 (  0% 完成): [MASK] [MASK] [MASK] [MASK] [MASK] [MASK] [MASK]...
t=0.950 (  5% 完成): [MASK] [MASK] [MASK] [MASK] [MASK] [MASK] [MASK]...
...
t=0.000 (100% 完成): 2015: $141,938 versus; 2016: $337,746. [49] On Sunday...
======================================================================
```

每行格式：`t={time} ({percent}% 完成): {decoded_text}`
- `[MASK]` = 未揭开的 token（吸收态）
- 其余文本 = 已确定的 token（BPE 子词已合并）
