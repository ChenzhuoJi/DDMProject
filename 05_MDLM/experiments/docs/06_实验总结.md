# 6. 实验总结

## 6.1 核心结论

本文围绕 Masked Diffusion Language Model（MDLM）开展了一系列实验，涵盖扩散过程可视化、采样算法效率对比与长文本生成等方面。实验覆盖了 MDLM 框架从原理验证到性能评测的主要环节，主要结论可归纳为以下两点。

第一，扩散模型的生成过程呈现清晰的阶段性特征。无论采样步数或随机种子如何变化，去噪行为均高度集中于反向过程的后半程——前 50% 的步数中序列几乎保持全掩码状态，后 50% 的步数中词汇逐步析出，而最后 10–20% 的步数完成精细调整。这一现象由 LogLinear 噪声调度的数学性质决定，也与连续时间马尔可夫链（CTMC）框架的理论预期一致。不同随机种子下生成文本的显著差异直观地展示了吸收态马尔可夫链的样本路径多样性。

第二，采样算法的选择对推理效率有显著影响，但不影响生成质量。ddpm_cache 采样器通过利用马尔可夫性缓存条件概率分布，在 1000 步配置下达到了 9.6 秒的端到端耗时，分别比 ddpm（31.8 秒）和 analytic（36.5 秒）快 3.3 倍和 3.8 倍。三种采样器在相同随机种子下生成完全一致的文本，这一等价性验证了 MDLM 框架下采样策略选择的可互换性。

## 6.2 实验现象与《随机过程》课程概念对照

| 实验现象 | 随机过程概念 | 具体对应 |
|---------|-------------|---------|
| t=1→0 逐步去噪 | 非时齐马尔可夫链 | 转移概率矩阵随时间变化，前向过程 q(zₜ\|x) |
| [MASK] 为吸收态 | 吸收态马尔可夫链 | 一旦进入 [MASK] 状态，在前向过程中不再离开 |
| 连续时间极限 T→0 | CTMC | 当离散步数趋于无穷时，马尔可夫链趋于连续时间过程 |
| LogLinear 噪声调度 σ(t) | 转移速率矩阵 | σ(t) 定义了生成元矩阵的时变参数 |
| ddpm_cache 缓存复用 | 马尔可夫性（无后效性） | 当前状态已知时，未来转移与历史路径无关 |
| 不同 seed → 不同文本 | 随机过程的样本路径 | 相同转移核下的不同轨迹实现 |

## 6.3 实验局限与改进方向

本文实验存在若干局限性。第一，本研究的推理实验全部基于单一预训练 checkpoint（`mdlm-no_flashattn-fp32-owt`），未对不同规模模型（如 `mdlm-owt` 的 fp16 版）或不同骨干网络（如 DiMamba）进行对比评测，模型的泛化能力与架构敏感性尚待进一步检验。第二，扩散过程可视化的步数对比实验仅涉及 128、500 与 1000 三种配置，更细致的离散化粒度扫描（如从 50 步到 2000 步的多点采样）将有助于更精确地刻画质量—效率曲线。
