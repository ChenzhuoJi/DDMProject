## 可视化数据（.txt）

扩散过程的逐行快照，展示从 `t=1.0`（全 MASK）到 `t=0.0`（完整文本）的去噪过程。

| 文件 | 说明 | 关键参数 |
|:----|------|:--------:|
| **snapshot_seed42.txt** | seed=42 去噪过程 | 128步, ddpm_cache, OWT |
| **snapshot_seed123.txt** | seed=123 去噪过程 | 同上 |
| **snapshot_seed999.txt** | seed=999 去噪过程 | 同上 |
| **snapshot_steps500.txt** | 500步采样对比 | seed=42, 500步 |
| **snapshot_text8.txt** | text8 tiny 模型生成 | 本地训练的 tiny 模型 |

### 文件格式

```
======================================================================
Diffusion Reverse Process (t=1.0 → t=0.0)
======================================================================
t=1.000 (  0% done): [MASK] [MASK] [MASK] [MASK] [MASK] [MASK] [MASK]...
t=0.898 ( 10% done): at brain [MASK] [MASK] [MASK] [MASK] [MASK]...
...
t=0.000 (100% done): at brain news no of made by split of hew...
======================================================================
```

每行格式：`t={time} ({percent}% done): {decoded_text}`
- `[MASK]` = 未揭开的 token（吸收态）
- 其余文本 = 已确定的 token

## 采样器效率日志（.log）

| 文件 | 采样器 | 耗时 | Gen. PPL |
|:----|:-----:|:---:|:--------:|
| **ddpm_cache.log** | ddpm_cache（缓存加速） | 17.5s | 2509.95 |
| **ddpm.log** | ddpm（标准 ancestral） | 22.6s | 2509.95 |
| **analytic.log** | analytic（SEDD 式） | 18.9s | **1929.50** |

### 文件内容结构

```
Seed set to 42
CONFIG ...              ← Hydra 配置树（模型、数据、采样参数）
[2026-...] Generating samples.
... FutureWarning ...   ← PyTorch API 弃用警告（可忽略）
Text samples: [...]     ← 生成的文本样本
Generative perplexity: tensor(X.XXXX)  ← GPT-2 Large 评估的生成困惑度
WALL_TIME_MS: XXXXX     ← 总耗时（毫秒）
DONE
```

### 关键字段

| 字段 | 含义 |
|:----|------|
| `sampling.predictor` | 采样器类型（ddpm_cache / ddpm / analytic） |
| `sampling.steps` | 采样步数（128） |
| `model.length` | 序列长度（256） |
| `Text samples` | 模型生成的文本 |
| `Generative perplexity` | GPT-2 Large 评估的困惑度（越低越好） |
| `WALL_TIME_MS` | 从启动到完成的毫秒数 |
